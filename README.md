PPP Currency Exchange

PPP Currency Exchange is an Android application that combines traditional live currency conversion with Purchasing Power Parity (PPP) calculations. Unlike most currency converter apps that only display market exchange rates, this app also helps users understand how much money is actually worth across different countries based on purchasing power.

For example, while USD 5,000 may convert to around ₹4–5 lakh using the market exchange rate, its PPP value in India may be significantly different because the cost of living and purchasing power are not the same. This makes the app useful for comparing salaries, living expenses, travel budgets, international income, and economic value across countries.

Features

- 🌍 Live currency conversion using online exchange rates
- 📶 Online and Offline modes
- 💱 Manual exchange rate support in Offline mode
- 📊 Purchasing Power Parity (PPP) converter
- 🔄 Currency swap functionality
- 💾 Offline caching of exchange rates
- 📱 Modern Material 3 UI
- 🌙 Light and Dark themes
- 📖 Educational About section explaining PPP and exchange rates
- ⚡ Fast, lightweight, and responsive design

Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- MVVM Architecture
- Hilt Dependency Injection
- Kotlin Coroutines
- Flow / StateFlow
- Room Database / DataStore
- REST API for live exchange rates

Installation

1. Download the latest APK from the Releases section.
2. Install the APK on your Android device.
3. If Android displays a security warning because the app is not installed from the Play Store, allow installation from this source.
4. Open the app and keep Online Mode enabled to automatically fetch the latest exchange rates.
